# Adidas 

This repo contains my Adidas Amsterdam assignment.

## 🛠 Installation

```bash
Clone the repo, open the project in Android Studio, hit "Run". Done!
Don't forget to set the API url in the AppModule.kt if the server doesn't work ;)
```
